# Showcase Trial v1

## 版本定位

这是展示仓的第一版公开试运行版本。

这个版本的目标，不是完整开放母仓全部能力，而是提供一版可展示、可试运行、可测试、可验证接入路径的公开版本。

---

## 本版本包含内容

当前版本已对外提供：

- 展示型 README 首页
- 母仓核心架构的试运行骨架
- `runbook.md` 主链说明
- `inputs / checks / frameworks / outputs / reviews / cases / prompts` 目录结构
- GitHub Actions 试运行工作流
- Trial Input Request / Bug Report / Feature Request issue 模板
- 版权说明

---

## 本版本不包含内容

当前版本不默认开放：

- 母仓 Python 智能核心
- 母仓完整自动化实现
- 母仓全部演进能力

也就是说：

**这个版本用于试运行验证，不等于母仓完整实现对外开放。**

---

## 适用场景

本版本适合：

- 对外展示产品方向
- 客户先行测试流程是否能跑通
- 验证结构、模板、试运行入口和接入方式
- 作为合作前的轻量验证载体

---

## 当前边界

当前采用的原则是：

- 母仓承载核心智能
- 展示仓承载试运行与展示
- 展示仓可运行，但不承担完整智能表现
- 先验证流程，再决定是否进入更深合作
